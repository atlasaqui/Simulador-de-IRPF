# Simulador-de-IRPF
Projeto desenvolvido como parte da disciplina de [Programação para Web] na Universidade Católica de Pernambuco (UNICAP). O objetivo é automatizar o cálculo do imposto de renda simplificado.

🛠️ Tecnologias Utilizadas
Front-end: HTML5, CSS3 (Design Responsivo)

Lógica: JavaScript (Cálculo de alíquotas e deduções)

Design: UI/UX focado em clareza de dados.

🚀 Funcionalidades
Cálculo automático baseado nas tabelas vigentes da Receita Federal.

Validação de campos em tempo real.
